# Evilbitb_Overview
### BITB (Browser In The Browser) Attack  **What It Is:** A fake browser window rendered INSIDE a real browser window.  **Components:** 1. Fake browser chrome (traffic lights, URL bar, lock icon) 2. Fake URL bar showing victim actual URL (polled via /get_url) 3. noVNC iframe with real Chrome VNC stream 4. Login popup overlaid on top
